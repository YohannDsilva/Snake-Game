This project is a Java Swing recreation of the classic Snake arcade game. The player controls a snake around a grid, eating food to grow longer and score points, while avoiding the walls and their own body. The program uses a Timer-driven game loop, custom Java2D rendering, and object-oriented design split across eight source files. The game runs in a 28x22 grid rendered inside a JFrame. Five red apples are always present on the board; eating one grows the snake, adds 10 points, and respawns it in a new free cell. Every 120 game ticks a golden apple appears worth 40 points, which blinks and disappears after 60 ticks if uncollected. The score and a persistent high score are shown in a HUD at the top. Difficulty can be set to Easy, Normal, or Hard before each round, changing the tick speed from 240 ms down to 100 ms. The game can be paused with P or Escape at any time. When the snake dies a short flicker animation plays, the snake alternates between red and near-black for seven frames while a red tint fades across the screen, before the Game Over overlay appears. Requires Java 16 or later. Download all eight .java files into one folder, then run: javac *.java followed by java SnakeGame. No external libraries or build tools are needed. Controls are arrow keys or WASD to move, P or Escape to pause, and Enter or Space to start or restart. The program satisfies all listed requirements. The snake moves with arrow keys and grows on food collection. Wall and self-collision both end the game. Score is tracked live and a high score persists across restarts. A Game Over screen displays the final score with a restart button. A Start button appears on the ready screen. The game loop uses javax.swing.Timer. The design is object-oriented with movement logic in Snake.java and all rendering in GamePanel.java.
